# acwmctl 

A basic command-line tool to control Intesis Wi-Fi module ( MH-AC-WIFI-1 ), controlled by AirConWithMe app. 

## Building 

Type `make install` to build binary and copy it to ~/bin/ directory. 

## Configuration 

create a file ~/.acwm

```json 
{
    "ip" : "192.168.1.123",
    "username" : "admin",
    "password" : "admin"
}
```
